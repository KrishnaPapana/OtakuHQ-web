OtakuHQ

Privacy Policy

Last updated: April 23, 2026

OtakuHQ is a Chrome extension that helps anime fans discover streaming platforms, track a full weekly airing schedule, save favorites, receive episode alerts, search titles, browse trending charts and genre rankings, and read anime news — all from one popup.

Information OtakuHQ does NOT collect

  • Personal account information or login credentials
  • Payment or financial information
  • Browsing history or visited URLs
  • Any data transmitted to the developer or any third-party server

Information stored locally in Chrome

All data OtakuHQ saves lives in chrome.storage.local on your device. It never leaves your browser.

ItemWhat it stores
Region preference Your selected region filter (All / Global / India / US)
Favorites Anime you have hearted — title, cover image URL, AniList URL, next-episode data
Notification preference Whether episode alert notifications are enabled (true/false)
Notified set Log of notification IDs already sent, capped at 200 entries, to prevent duplicate alerts
Unofficial domain cache Resolved working URLs for fan streaming sites, refreshed each time you open the popup

External data sources accessed

SourcePurpose
graphql.anilist.co Anime schedule, releases, trending, search, and genre rankings
news.google.com Anime news RSS headlines, locale-matched to your region
Fan-site seed URLs HTTP fetch used only to follow redirects and find the current working domain — no page content is scraped or stored
google.com/s2/favicons Favicon images for the streaming platform icon strip

All requests are read-only. No user data is included in any outbound request.

How data is used

OtakuHQ uses data solely for its single purpose:

  • Displaying anime platforms, schedule, releases, trending, search results, and news inside the popup
  • Restoring your saved region preference and favorites across browser sessions
  • Sending local Chrome notifications 15 minutes before a favorited episode airs
  • Showing a badge counter on the toolbar icon for favorites airing today
  • Resolving current working domains for fan streaming sites

OtakuHQ does not use data for advertising, profiling, analytics, or any purpose unrelated to its single purpose. OtakuHQ does not sell user data.

Permissions justification

PermissionJustification
storage Store region, favorites, notification preference, domain cache, and sent-notification log locally on your device
alarms Run a 1-minute periodic background check to fire episode alerts and update the toolbar badge counter
notifications Display Chrome push notifications when a favorited episode is about to air
https://graphql.anilist.co/* Fetch all anime data (schedule, releases, trending, search, genres) from AniList's public API
https://news.google.com/* Fetch the anime news RSS feed
<all_urls> Follow HTTP redirects from fan-site seed URLs to discover current working domains. OtakuHQ does not read or modify any page you visit.

Data sharing

OtakuHQ does not share any user data with third parties. All outbound network requests are direct, read-only calls to public APIs and contain no personally identifiable information.

Data retention

  • Region preference — until you change it or uninstall the extension.
  • Favorites — until you remove each favorite or uninstall the extension.
  • Notification preference — until you toggle it off or uninstall.
  • Notified set — auto-trimmed to 200 most recent entries; purged on uninstall.
  • Unofficial domain cache — refreshed on each popup open; purged on uninstall.

Security

  • OtakuHQ uses the minimum permissions required for its functionality.
  • It does not inject scripts into web pages.
  • It does not execute remote code.
  • It does not require user sign-in or account creation.
  • All data processing happens locally within the extension.

Children's privacy

OtakuHQ does not knowingly collect any information from children. Adult content is explicitly excluded from all AniList API queries (isAdult: false).

Changes to this policy

If data practices change, this policy will be updated with a new date. Continued use of the extension after an update constitutes acceptance of the revised policy.
